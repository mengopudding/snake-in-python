# SNAKES 

My intro to Python with a childhood memory game

## How to play

- Have `python2` installed

- Clone this repo

  `https://github.com/mengopudding/snakes.git`
- To start playing

  `python snake_game.py`

- To quit, press any key other than `arrow keys`
